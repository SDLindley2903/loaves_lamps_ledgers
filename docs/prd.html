What we're building

A companion, not a clone

The grown-up reader stays exactly as it is. This is a new, brightly-colored experience for children — with its own new name — that turns your three big ideas into things kids can watch, hear, and play.

  • 🗺️
    A friendly "world" screenThree places to visit — the Bakery, the Lamp, and the Treasure Chest — one per big idea.
  • 🐭
    An animated guide characterA mascot that talks, cheers, and explains — so no reading is required.
  • 🔊
    Read-aloud everywhereTap any words and the app speaks them, using the device's built-in voice.
  • 🎮
    Three mini-gamesBread Line (work), Light the Path (tracking), and The Three Jars — Give / Save / Spend.
  • ⭐
    Sticker rewardsFinish a story or game, earn a sticker for the sticker book.
  • 🔒
    A calm Parent ZonePIN-gated notes and settings, so parents and teachers trust it.

Quality · light to heavy

How we test

Each gate is a bigger safety check than the last. We use Playwright (with the CRX Chrome extension) against the app's web build, plus Pytest for any backend later.

GateQuestion it answersWhen
1 · SmokeDoes the page load with no errors?Every phase
2 · FunctionalDoes the feature work (with fakes)?Every game
3 · IntegrationDoes it work against real servers?Phase 2+
4 · QA gateQuality, errors, database, speedBefore releases
5 · Security gateInput, login, data protectionBefore any accounts
